## 📱 Responsive Design

El proyecto está diseñado para ser 100% responsive, adaptándose correctamente a los siguientes dispositivos:

- 📱 Móviles (320px - 480px)
- 📱 Tablets (768px)
- 📱 Tablets Horizontales
- 💻 Escritorios (1024px en adelante)
